<h2>QQ 美化</h2>
<p>==特别推荐！==<br/>
QQ 是大部分中国人不得不用又恨之入骨的东西——广告、弹窗、不想看的娱乐页面、花里胡哨功能性差的界面。<br/>
Windows 端自从推出 QQ 9，上述情况改善了不少，然而恶心人的东西最近逐渐加回来了，只能说本性难移。<br/>
为了不被迫在这个常用软件上天天吃屎，伟大的具有开源精神和折腾精神的中文互联网人纷纷投入 QQ 改造计划，而今天我要介绍的就是其中一个伟大项目——<strong>LiteLoaderQQNT</strong>。<br/>
好了说正事，liteloader 是 QQNT 的插件平台，安装后可以下载众多插件。<br/>
GitHub 项目地址：<a href="https://github.com/LiteLoaderQQNT/LiteLoaderQQNT">LiteLoaderQQNT: QQNT 插件加载器：轻量 · 简洁 · 开源 · 福瑞</a></p>

<ol>
<li><a href="https://github.com/xinyihl/LiteLoaderQQNT-PluginInstaller/tree/main">PluginInstaller</a>：LiteLoaderQQNT 插件安装器，可检查更新和一键安装/重启。先安装这个再安装其他插件省事很多。</li>
<li><a href="https://github.com/ltxhhz/LL-plugin-list-viewer/tree/main">LL-plugin-list-viewer</a>: LiteLoaderQQNT Plugin 插件列表查看·安装·更新。收录了大部分插件，可直达 GitHub 项目地址。安装功能存在问题，部分插件需要手动安装和更新，否则会报错无法启动 QQ，建议作为插件/主题查看器。</li>
<li><a href="https://github.com/xiyuesaves/LiteLoaderQQNT-lite_tools/tree/v4">轻量工具箱 —— 轻量 · 优雅 · 高效 · 福瑞</a>：聚合了大量功能的工具箱，免于四处寻找插件。部分可选功能如下：<ul>

<li>选项高亮，特殊消息高亮</li>

<li><strong>消息预览</strong>：根据消息中第一个链接生成一张类 tg 的预览卡片</li>
<li>本地表情</li>
<li>消息后缀</li>
<li><strong>撤回消息缓存并高亮</strong></li>
<li><strong>设置背景</strong>，调整亮度、透明度，可加磨砂等效果</li>
<li><strong>精简侧边栏</strong>，所有功能可开关</li>
<li>输入框、消息框功能开关</li>
<li>……</li>
</ul>
</li>
<li><a href="https://github.com/xh321/LiteLoaderQQNT-QR-Decode/tree/master">二维码解析</a>：对 QQNT 聊天中的图片进行二维码解析</li>
<li><a href="https://github.com/xh321/LiteLoaderQQNT-Directly-Jump/tree/master">使用自定义浏览器打开链接并跳过拦截页</a></li>
<li><a href="https://github.com/MUKAPP/LiteLoaderQQNT-DeepL/tree/main">将 DeepL 翻译接入你的 QQNT</a></li>

<li><a href="https://github.com/xh321/LiteLoaderQQNT-Kill-Update/tree/master">Kill-Update</a>：关闭 QQ自动更新弹窗，有些插件不会第一时间支持最新版 QQ，所以禁止更新是有用的。</li>

<h2>Wallpaper Engine</h2>
<p>==必备好物！==<br/>
鼎鼎大名的万能小红车，开启××√享受崭新人生（bushi）<br/>
作为最常用最好用的壁纸软件，wallpaper 有许多优点：</p>
<ul>
<li>资源丰富。steam 创意工坊每时每刻都有大量优质壁纸上架，且几乎全部免费下载使用。</li>
<li>资源种类多。视频、图片、动图、网页……壁纸种类极多，且不少壁纸功能丰富，聚合了音乐歌词、特效、频谱等功能，一键装点桌面。</li>
<li>找资源方便。Wallpaper 搜索和筛选规则完善丰富，可筛选壁纸分辨率、类型、适用年龄等。</li>
<li>使用简便。基于 steam 创意工坊，能访问 steam 就能下载壁纸，无需翻墙，下载速度有保障。大部分壁纸，即使功能复杂，在 wallpaper 界面也能轻松设置。</li>
<li>串联手机。Wallpaper 推出了安卓应用，可通过 PC 向手机传输使用壁纸。</li>
</ul>
<p><strong>Warning</strong>：部分复杂特效网页、高清视频壁纸较为消耗 GPU 性能，占显存较多。可以在 wallpaper 设置里调整帧率、特效和应用行为等改善。</p>
<p>唯一的缺点是在 steam 购买 wallpaper 需要 19 RMB，不过这价格真不高吧。</p>
<p>Steam：<a href="https://store.steampowered.com/app/431960/Wallpaper_Engine/">Steam 上的 Wallpaper Engine：壁纸引擎</a></p>

<h2>Sapphire</h2>
<p>==特别推荐！==<br/>
桌面图标、布局美化，<strong>更改交互体验</strong>。</p>
<ul>
<li>可调整桌面图标布局，更改横纵网格数</li>
<li>放置格子（类手机上的文件夹），将桌面各类文件直接放进文件夹，分类同一管理</li>
<li>调整图标、格子大小（网格数整数倍），圆角，背景，透明度</li>
<li>更改文件名的字体</li>
<li>精简模式去除文件名，可每个图标单独设置</li>
<li>可设置交互动效</li>
<li><strong>Dock 栏</strong>，类 Mac/手机下方的 dock，可竖置，改变长宽、背景色</li>
<li>开启文件夹高级交互后可以文件夹鼠标悬停预览内容，单击原地展开，不必开文件资源管理器窗口。</li>
<li><strong>针对 steam 内应用的快捷方式优化</strong></li>
<li><strong>双击隐藏图标</strong>，每个可单独设置是否隐藏。</li>
<li>后台内存占用 100 mb 左右，尚可接受。几乎不消耗 GPU</li>
<li>可自定义图片、动图作为壁纸。初步兼容 WallpaperEngine（需在设置中设置当其他程序成为焦点时为始终运行）。</li>
<li>多屏支持</li>
</ul>
<p>缺点：</p>
<ul>
<li>我的电脑开启高级文件交互后，只要文件夹内文件数稍多，sapphire 会直接卡死。观望作者更新。</li>
<li>暂时还没有保存桌面布局的功能，好在即使卡退也会记录最新布局。</li>
<li>只保证兼容 Windows 11</li>
<li>更改桌面右键交互但目前还不能自定义</li>
</ul>
<p>GitHub 项目地址：<a href="https://github.com/hymnly133/Sapphire-EnhancedDesktop">Sapphire-EnhancedDesktop: Windows桌面取代软件</a></p>
