<li>下载<br>从官网下载node.js后，基本只需要一路next即可，最重要的是选择add to path.</li>
<li>使用源<ul>
<li><p>截止到2024&#x2F;2&#x2F;19为止，网上很多教程中的淘宝源均已无法使用，下面的为最新且有效的淘宝镜像:<a href="https://registry.npmmirror.com/">https://registry.npmmirror.com</a></p>
</li>
<li><p>先下载好cnpm会让之后的工作事半功倍b（￣▽￣）d</p>
</li>
</ul>
</li>
</ul>
<span id="more"></span>
<p>下面贴上代码:</p>
<pre><code class="hljs node.js">
npm i -g cnpm --registry=https://registry.npmmirror.com</code></pre>

<ul>
<li>目录权限问题：如果遇到npm下载报错说权限不够等问题，win + r 输入cmd之后不着急回车，此时ctrl+shift+回车即可以管理员身份打开cmd，即可解决问题。</li>
</ul>
<hr>
<h2 id="hexo相关"><a href="#hexo相关" class="headerlink" title="hexo相关"></a>hexo相关</h2><ul>
<li><p>如果成功的下载了cnpm，那么之后使用npm下载的东西都用cnpm来替代，就能解决大部分无法下载的问题(๑•̀ㅂ•́)و✧</p>
</li>
<li><p>创建一个专门处理blog的文件夹，之后所有操作都在这个文件夹的根目录进行即可。</p>
</li>
<li><p><em><strong>重点</strong></em>：hexo init报错: 此时只需要在根目录输入:</p>
</li>
</ul>
<pre><code class="hljs cmake">
cnpm <span class="hljs-keyword">install</span>
hexo s
</code></pre>

<p>即可解决。</p>
