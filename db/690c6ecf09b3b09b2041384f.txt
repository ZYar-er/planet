<h2 id="关于我校一站式服务大厅">关于我校一站式服务大厅</h2>
<p>我们学校的一站式服务大厅是由南京金智教育开发的，而且很多学校都用这个系统。所以，接下来的部分，感觉很多同志们都能适用。</p>
<p>我们学校的一站式大厅有很多功能，不过其中仅仅有很少的一部分我们能用。我感觉平常使用最多的就是：</p>
<ul>
<li>课表</li>
<li>成绩查询</li>
<li>考试安排</li>
</ul>
<p>而我们平常要使用一站式(或者其他东西，比如健康报告)的话，一般都要先登陆统一认证平台，之后才能进入一站式。而这个统一认证平台，也是金智的:-P</p>

<h2 id="如何登陆统一认证平台">如何登陆统一认证平台</h2>
<h3 id="工具">工具</h3>
<p>根据 <a class="link" href="https://github.com/xdlinux/xidian-scripts"  target="_blank" rel="noopener"
    >xidian-script</a> 姐妹计划 <a class="link" href="https://github.com/xdlinux/libxduauth"  target="_blank" rel="noopener"

</span></span></code></pre></div><h4 id="加密密码">加密密码</h4>
<p>首先，我们需要填充密码。这里我们使用 PKCS7 填充方式。我的程序自己实现了，因为没看明白 Dart 里面 PKCS7 咋用的:-(：</p>
<ul>
<li>由于加密方式是 AES，要求分组长度是 128 bytes，根据 1 bit = 8 bytes，需要字符串长度是 16 的倍数。</li>
<li>先在密码字符串前面插上 xidianscriptsxdu 四遍(正好是 16 长的字符串，Dart 默认给的是随机字符串)，然后将字符串转换成 int 数组(也就是把每个字符转换成对应的 ASCII 码)。</li>
<li>然后计算我们还需要在后面插入多少元素来满足 16 的倍数，差几个在后面插入几个。插入内容为插入元素的个数。注意，要是一个都不差，也得插入数据，来保证数据和插入值都存在。</li>
<li>由于接下来加密需要的是数字数组，所以不变回字符串。</li>
</ul>
<p>然后，从 form 里面寻找 input 标签，且 id 为 pwdEncryptSalt 的元素，这里面是加密密码的密钥(还是盐啥的)。最后，使用 AES-CBC 算法加密，然后返回字符串。</p>

</span></span></code></pre></div><h4 id="发送登陆报文">发送登陆报文</h4>
<p>首先，构建登陆请求所需要的头。初步需要三个信息：</p>
<ul>
<li>username：你的学号</li>
<li>password：加密过的密码</li>
<li>rememberMe：防止之后又走一遍登陆过程</li>
</ul>
<p>在此之后，我们需要寻找很多预配置的头，加入到我们的头中。这些元素都在 form 中，是 input 标签 ，参数是 <code>type=hidden</code> 。循环将其插入到请求头中。</p>
<p>最后，向 <a class="link" href="http://ids.xidian.edu.cn/authserver/login"  target="_blank" rel="noopener"
    >http://ids.xidian.edu.cn/authserver/login</a> 发送 POST 请求，其中请求头是上面搞过的请求头，请求参数是 <code>{&quot;service&quot;: 接下来要访问的网址}</code> ，请求不要自动跳转。</p>
<p>如果正常，ids.xidian.edu.cn 下面的 Cookie 会多一个 <code>happyVoyagePersonal=...; Path=/personalinfo</code>。然后需要我们自己向需要跳转的新网址发 GET 请求，同样，不需要自动跳转。</p>
<p>到此，通过统一认证平台，我们登陆了一站式服务大厅。</p>
